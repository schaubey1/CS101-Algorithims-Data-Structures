List.java
-----------------------------------------------------------------------------
List.java is used to construct a List ADT using a sequence of finite integers. 
It is used to store an array of strings from the provided input files and perform 
manipulations on them.
-----------------------------------------------------------------------------

Lex.java
-----------------------------------------------------------------------------
Lex.java is the main program that is utilized to implement the List ADT in order
to test the functions of sorting an array, reading strings, and applying InsertionSort
to store the indices in a sorted order. It saves the manipulated array to a file after operation completes.
-----------------------------------------------------------------------------

File: ListClient.java
-----------------------------------------------------------------------------
Instructor provided test client for the List ADT.
-----------------------------------------------------------------------------

File: Makefile
-----------------------------------------------------------------------------
Makefile is utilized to create the Lex.jar executable file by combining all the files.
---------------------------------------------------------------------------
